# Flixter

A two-sided video-streaming marketplace platform.

# Features

- Built on Rails
- Mobile-responsive
- Credit card payment capabilities (via the Stripe API)
- User role management (e.g. instructors and students)

# Tools Used

- Ruby on Rails
- PostgreSQL
- Bootstrap
- Devise
- Carrierwave
- jQuery
- VideoJS
